Wawona

System Utilities Repo

Wawona is a high-performance Wayland compositor for iOS, macOS, and Android. This repository supports iOS 13.0 - 26.0+ and provides specialized jailbreak tweaks (Springboard/LockScreen replacements, DRM/dmabuf support) and native Wayland software.
