Wawona

System Utilities Repo

Wawona is a high-performance Wayland compositor for iOS, macOS, and Android. This repository supports iOS 13.0 - 26.0+ (Rootless/RootHide) and Android (Termux), providing specialized software and native Wayland utilities.

Manual iOS Setup

Option A: Sileo / Zebra

Open Sileo or Zebra, go to Sources, tap "Edit" → "Add", and enter the URL:

https://repo.wawona.io/
CLI Setup

The CLI setup is identical for both platforms. On iOS, use NewTerm. On Android, use Termux.

Option B: Manual Editor

Run sudo apt edit-sources (iOS) or apt edit-sources (Android) and add:

deb https://repo.wawona.io/$( [ "$(uname -o)" = "Android" ] && echo "android" || echo "" ) stable main

Note: Use /android suffix only on Termux.

Option B: One-click CLI

curl -sL https://repo.wawona.io/setup.sh | bash
