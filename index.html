Wawona

System Utilities Repo

Wawona is a high-performance Wayland compositor for iOS, macOS, and Android. This repository supports iOS 13.0 - 26.0+ (Rootless/RootHide) and Android (Termux), providing specialized software and native Wayland utilities.

Recommended for your device
iOS Setup (Rootless) RECOMMENDED

Option A: Sileo / Zebra

https://repo.wawona.io/

Option B: Manual Editor

Run sudo apt edit-sources and add:

deb https://repo.wawona.io/ stable main

Option C: Direct Command

echo "deb https://repo.wawona.io/ stable main" | sudo tee /etc/apt/sources.list.d/wawona.list && sudo apt update
Termux Setup (Android) RECOMMENDED

Option A: One-click CLI

echo "deb https://repo.wawona.io/android stable main" > $PREFIX/etc/apt/sources.list.d/wawona.list && apt update

Option B: Manual Editor

Run apt edit-sources and add:

deb https://repo.wawona.io/android stable main
